# My Games
A collection of simple and interactive games built using Python.
This repository is created for learning, practice, and showcasing beginner-level projects.

Each game is independent and can be run separately.

***
## tic-tac-toe-game
A simple Tic Tac Toe game built using Python and Tkinter.
The AI uses the Minimax algorithm.

## How to run
1. Install Python 3
2. Download or clone this repository
3. Run:
   python tic-tac-toe.py

***
## rock-paper-scissors-game

## flappy-bird-game
A classic game of flappy bird with basic features built using pygame.

## How to run
1. Install Python 3, pygame (install via pip) (make sure to use python 3.12.x, as pygame doesn't support higher versions)
2. Download or clone this repository
3. Navigate to the flappy-bird-game folder.
4. Run:
   python flappy-bird.py

## controls
1. Spacebar: Flap the bird
2. ESC / Close window: Quit the game
