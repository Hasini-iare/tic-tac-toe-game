# My Games
A collection of simple and interactive games built using Python.
This repository is created for learning, practice, and showcasing beginner-level projects.

Each game is independent and can be run separately.

***
## tic-tac-toe-game
A simple Tic Tac Toe game built using Python and Tkinter.
The AI uses the Minimax algorithm.

## How to run
1. Install Python 3
2. Download or clone this repository
3. Run:
   python script.py

***
## rock-paper-scissors-game
A simple rock paper scissors game built using python and Tkinter.
The AI uses the Minimax algorithm & pattern-based decision making.
Score tracking.

## How to run
1. Install Python 3
2. Download or clone this repository
3. Run:
   python rock-paper-scissors.py
